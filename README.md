# wall
observer-wall
Этот репозиторий содержит несколько текстов, созданных независимо друг от друга.
Все они описывают один и тот же объект разными словами.
Не предполагается, что читатель согласится с текстами.
Не предполагается, что они содержат инструкции.
Они сохранены здесь как метки наблюдения.
Если какая-либо структура окажется узнаваемой, этого достаточно.
